# Images Directory

Store all your images here for optimal organization and SEO.

## Image Guidelines

### Dimensions
- **Featured Images**: 1200x630px (optimal for social sharing)
- **In-post Images**: 800px-1200px wide
- **Thumbnails**: 500x500px or 500x750px
- Keep aspect ratios consistent across similar image types

### File Size
- **Target**: Under 200KB per image
- **Maximum**: 300KB
- Use compression tools like TinyPNG, ImageOptim, or Squoosh

### File Naming
- Use descriptive, SEO-friendly names
- Use hyphens, not underscores: `ai-tools-2026.jpg` not `ai_tools_2026.jpg`
- Include keywords where relevant
- Use lowercase

### Alt Text (Critical for SEO!)
Always add descriptive alt text in your markdown:
```markdown
![Descriptive alt text with keywords](/images/your-image.jpg)
```

### Supported Formats
- **JPEG**: For photos and complex images
- **PNG**: For graphics with transparency
- **WebP**: For modern browsers (best compression)
- **SVG**: For logos and icons

### Organization
Consider organizing by post or category:
```
images/
  ├── logo.png
  ├── profile.jpg
  ├── posts/
  │   ├── ai-tools-2026.jpg
  │   └── productivity-tips.jpg
  └── thumbnails/
      └── ...
```

## Required Images

### Immediate:
1. **logo.png** - Your blog logo (transparent background recommended)
2. **profile.jpg** - Your photo for the About Me page
3. **favicon.ico** - Browser tab icon (16x16, 32x32, 48x48)

### For Each Post:
- One featured image (1200x630px)
- Supporting images as needed (optimized)

## Image Optimization Tools

- **TinyPNG**: https://tinypng.com
- **Squoosh**: https://squoosh.app  
- **ImageOptim** (Mac): https://imageoptim.com
- **GIMP** (Free): https://www.gimp.org

## SEO Benefits of Proper Images

1. **Google Image Search** - Properly optimized images with alt text can rank independently
2. **Page Speed** - Compressed images improve load times and SEO rankings
3. **User Experience** - Fast-loading images reduce bounce rates
4. **Social Sharing** - Proper featured images improve click-through rates
5. **Accessibility** - Alt text makes your content accessible to all users

Remember: Every image is an opportunity for additional search traffic!
